By caveshao

发表于 2019-03-27

文章目录

一 如何配置

本次教程中使用的 NAT 连接方式，具体步骤为

  1. 在系统上右击——>可移动设备——>网络适配器——>设置
    然后如下图选择 NET

    然后点击确定。

  2. 编辑——>虚拟网络编辑器——>右下角点击「更改设置」（有时候可能没有这个选项，跳过这一小步即可）

    先点击 NET 模式，然后将使用本地 DHCP 服务的选择为未勾选状态 。
    点击 NAT 设置，查看子网 IP 、掩码及 网关

  3. 启动系统，配置 centos 系统内的网络配置：
    输入 cd /etc/sysconfig/network-scripts
    回车，然后输入 ls 查看该目录下的文件

    找到与如图所示以 ifcfg-ens 开头的文件
    输入 su 转到 root 用户
    我的文件是 ifcfg-ens33
    所以我输入vim ifcfg-ens33 编辑该文件
    找到如下几个配置项，配置相应属性，具体属性为 NET 设置中查到的属性。
    ip 地址为子网 IP 随意改掉最后一个数

    ONBOOT=yes 
BOOTPROTO=static 静态ip
IPADDR=192.168.109.7 ip地址
NETMASK=255.255.255.0 子网掩码
GATEWAY=192.168.109.2 网关

  4. 修改 DNS

    vim /etc/resolv.conf

    修改 nameserver 为网关地址

    nameserver 192.168.109.2

如此，网络配置完成。
ping 宿主机测试一下。
下一篇文章我将具体介绍如何测试配置正确。
