win10 远程桌面验证错误

By caveshao

发表于 2022-06-09

文章目录

win10 更新后在使用远程桌面功能后会出现如下名叫身份验证错误的提示：

这是远程桌面相关的补丁更新造成的，不同的 windows 版本需要用到不同的方法解决这个问题。

对于 windows 专业版，可以直接修改 CredSSP 加密 Oracle 修正，具体步骤是 win+r -> 运行 gepdit.msc -> 计算机配置 -> 管理模板 -> 系统 -> 凭据分配 -> 加密Oracle修正 -> 上栏选择启用，下栏选择易受攻击 -> 应用（若无效则重启计算机)

在 windows 家庭版中，无法运行 gepdit.msc，可以用修改注册表的方法解决这个问题 win+r
-> 运行 regedit 打开注册表编辑器
-> 找到目录：计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System\
-> System 上右键新建项，名称 CredSSP
-> 在 CredSSP 上新建项，名称 Parameters
-> 在 Parameters 上新建 DWORD(32) ，名称AllowEncryptionOracle
-> 双击 AllowEncryptionOracle 设置值为 2 十六进制
设置成功后，远程桌面就不会报错了，如果没有效果重启电脑试一下。
